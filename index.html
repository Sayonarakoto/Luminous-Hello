What is GEN-C?

GEN-C transforms manual campus processes into a digital workflow. Students submit gate passes online, faculty approves instantly, and security verifies in real-time — fully paperless.

Features

1. Student Pass Management (Delegation-Aware)

Latecomer Pass System

Students submit requests for being late. Automatically assigned to the department's current effective approver.

Gate Pass System

Students request permission to leave campus. Routed to the effective approver for approval or rejection.

Special Pass System

Multi-day or special-permission passes with a fully "delegation-aware" approval chain.

2. Hybrid HOD Authority & Delegation

Temporary Delegation

Original HOD can transfer approval power to any faculty member for a set duration.

Permanent Role Transfer

Complete handover of the Original HOD title and all permanent powers to a successor.

Manual Revocation

Original HOD can instantly reclaim power from a Temp HOD at any time.

Automatic Expiry (Cron Job)

Background process automatically returns power to Original HOD when temporary delegation expires.

3. Dynamic, Real-Time Access Control

"Owner's Key" (isOriginalHodGuard)

Security check allowing only the permanent Original HOD to access delegation management.

"Approver's Key" (isEffectiveApproverGuard)

Protects all pass approval/rejection buttons, allowing only the current effective approver.

4. Auditing & Statistics

Immutable Delegation History

Logs every authority change (Delegate, Revoke, Auto-Revert, Permanent Transfer) to a history collection for auditing.

Department Statistics Dashboard

Current Effective Approver can access a dashboard with statistics (e.g., total passes, common reasons) for their department only.

5. "Smart" Adaptive User Interface

Role-Specific Delegation Page

UI adapts based on user's role: Original HOD sees management forms, buttons, and history; Temp HOD sees a simple "Status Box"; Regular Faculty blocked.

Real-time UI Refresh

Frontend automatically re-checks the user's authority status, ensuring the UI updates promptly if a delegation expires or is revoked.
